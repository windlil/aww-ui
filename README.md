## Features
- Support for on-demand import
- Support tree shaking
- Uses TypeScript, type tips good
- Contains commonly used components
- Compact installation package 

## Installation
```bash
#npm
npm i aww-ui -D

#yarn
yarn add aww-ui -D

#pnpm
pnpm add aww-ui -D
```
