## 特性
- 支持按需导入
- 支持树摇，打包体积小
- 使用TypeScript，类型提示不错
- 包含常用组件
- 小巧的包

## 安装
```bash
#npm
npm i aww-ui -D

#yarn
yarn add aww-ui -D

#pnpm
pnpm add aww-ui -D
```
